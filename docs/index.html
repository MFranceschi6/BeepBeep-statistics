    Beep Beep Statistics Webservice v0.1.0

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    This is a statistics web server for the Beep Beep application for retrieving statistics about your runs in a visual graphical format. For more information, visit the Github repository: https://github.com/MFranceschi6/BeepBeep-statistics

    GET /users/{user_id}/statistics/{statistics_name}

    By accessing this page and providing an existing user ID, you retrieve a set of 7 arrays about the temporal statistics of your runs for the following attributes, respectively: 'distance', 'average speed', 'average heartrate'. 'total elevation gain', 'elapsed time', as well as the run names and the run ids.

    Parameters

    Parameter In Type Required Description
    user_id path integer true The id of the user whose statistics you want to access
    statistics_name path string true Based on the statistics name passed, an array containing the run names corresponding to the statistics name supplied is returned. 'distances' = distance array; 'average_speeds' = average speed array; 'average_heart_rates' = average heartrate array ; 'elevation_gains' = total elevation gain array; 'elapsed_times' = elapsed time array.
